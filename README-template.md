### What I learned

I learned how to use the css grid property.

```css
.proud-of-this-css {
  .grid-container {
    display: grid;
    justify-self: center;
    align-self: center;
    grid-template-areas:
      'daniel daniel daniel daniel jonathan kira'
      'jeanette patrick patrick patrick patrick kira';
    grid-gap: 20px;
    padding: 80px;
    margin: 40px;
    background-color: hsl(210, 46%, 95%);
}
}
```
